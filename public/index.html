Palette Picker API

This RESTful API is built with NodeJs, Express, Knex and PostgreSQL. The data provided consists of palettes of hex codes for varying projects.

Endpoints:

GET all projects

https://palettepicker-be.herokuapp.com/api/v1/projects

GET all palettes

https://palettepicker-be.herokuapp.com/api/v1/palettes

Get palettes for specific project

Required: ID

https://palettepicker-be.herokuapp.com/api/v1/projects/ID/palettes

GET one project

Required: ID

https://palettepicker-be.herokuapp.com/api/v1/projects/ID

GET one palette

Required: ID

https://palettepicker-be.herokuapp.com/api/v1/palettes/ID

Search projects and palettes

https://palettepicker-be.herokuapp.com/api/v1/search

POST a new project

https://palettepicker-be.herokuapp.com/api/v1/projects

Body Format

{ name: }

POST a new palette

https://palettepicker-be.herokuapp.com/api/v1/projects

Body Format

{ name: , color_one: , color_two: , color_three: , color_four: , color_five: , }

PATCH a project

Required: ID

https://palettepicker-be.herokuapp.com/api/v1/projects/ID

Body Format

{ name: }

PATCH a palette

Required: ID

https://palettepicker-be.herokuapp.com/api/v1/palettes/ID

Body Format

{ name: , color_one: , color_two: , color_three: , color_four: , color_five: , } (use only params being patched)

DELETE a project

Required: ID

https://palettepicker-be.herokuapp.com/api/v1/projects/ID

DELETE a palette

Required: ID

https://palettepicker-be.herokuapp.com/api/v1/palettes/ID
